# LAMP-installer
Quick LAMP installer for Linux.

# Supported OS
- [x] Ubuntu 14
- [x] Ubuntu 16
- [x] Debian 8
- [x] Debian 9

# Installation
``` wget https://raw.githubusercontent.com/Skeid9/LAMP-installer/master/lamp.sh && chmod 777 lamp.sh ```

# Usage
```
[!] LAMP installer v1.0
[!] Usage: ./lamp.sh
```
